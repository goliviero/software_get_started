Introduction
============

Generate a pdf document from  a latex source file. Several application
can do it

.. code:: sh

   $ pdflatex cv_en.tex
   $ latexmk -pdf cv_en.tex
..

Clean up temporary TEX files created for a specific TEX file:

.. code:: sh

   $ latexmk -c
..
